# Skoli website
## Prequisites
- npm
- bower

## Install
All you need to install are npm & bower dependencies:
```sh
npm install
bower install
```

## How to run
We use gulp to compile static files & serve a local version of the website.
```sh
gulp serve
```
## How to deploy
```sh
gulp deploy
```
